# 🤝 Contributing

Thanks for helping improve `se-claude-skill`.

## 🐞 Reporting bugs

If you run into a problem, please open a **GitHub issue** with:

- what you were trying to do
- what you expected to happen
- what actually happened
- steps to reproduce the problem
- any relevant file names or screenshots

Please remove any personal, private, or sensitive information before posting.

## 🔀 Pull requests

Pull requests are welcome.

A few guidelines:

- keep PRs focused and reasonably small
- open an issue first for larger changes when possible
- update docs when behavior or structure changes
- avoid committing private notes, credentials, or personal data

The repository owner (`@Godimas101`) is the default reviewer for changes, and pull requests should be reviewed before merge.
